1. Add a New Student
Method: POST
URL: http://localhost:3000/students
Body (JSON):
{
  "name": "Neeraj Kumar",
  "roll_no": "103",
  "class": "Final Year" 
}

Output:- 
{
  "message": "Student added successfully",
  "id": 1
}


2. Enter Subject Marks
Method: POST
URL: http://localhost:3000/marks
Body (JSON):
{
  "student_id": 1, 
  "subject": "Mathematics",
  "score": 85,
  "max_score": 100
}

Output:- 
{
  "message": "Marks added successfully"
}


3. Get Individual Result (Report Card)
Method: GET
URL: http://localhost:3000/students/1/result
Logic: Calculates the weighted percentage. A student passes if the total percentage is >=40\%.

Output:- 
{
    "total": "156",
    "max": "200",
    "percentage": "78.00",
    "status": "Pass"
}
